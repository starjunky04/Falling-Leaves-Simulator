# Forest scene project: 

## Description:
This was a project for one of my university course modules where I had to create a simulation of a forest scene with falling leaves. There is a wind force which can be intensified by the user to blow the leaves off of their original path.
Based on the size of the leaves and animals seen, the velocity at which they move also changes.
There is also a tire swing which I chose to include as an indicator for how strong the wind is.

## How it works:
- Press r to restart the simmulation
- hold the mouse button down to add a stronger gust of wind
- when a stronger gust of wind is applied, the animals will move faster with the wind, the leaves will be blown further and the tire swing will also be moved with the wind
